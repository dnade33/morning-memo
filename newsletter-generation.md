# Newsletter Generation Skill — Morning Memo

## Overview
Each subscriber gets a personalized newsletter generated by Claude Haiku based on:
1. Their selected topics and deep-dive preferences
2. Fresh news stories fetched from RSS feeds that morning
3. A closing quote in their chosen style

## Prompt Template
```javascript
function buildPrompt(subscriber, stories) {
  return `You are the editor of Morning Memo, a warm and thoughtful daily newsletter.
Write a personalized morning newsletter for ${subscriber.first_name}.

Their interests: ${subscriber.topics.join(', ')}
${subscriber.city ? `Their city: ${subscriber.city}` : ''}
${subscriber.extra_notes ? `They told us: "${subscriber.extra_notes}"` : ''}

Today's stories to cover:
${stories.map((s, i) => `${i+1}. ${s.title}\n${s.summary}`).join('\n\n')}

Write the newsletter in this exact format:
- Warm greeting using their first name
- 5-6 story summaries, each with a bold headline and 2-3 sentence summary in plain English
- A closing quote in the "${subscriber.quote_style}" style — attribute it properly
- Sign off as "Your Morning Memo Editor"

Tone: warm, clear, no jargon. Written for someone who wants to feel informed, not overwhelmed.
Do not include any HTML — plain text only. The sending system will handle formatting.`
}
```

## Caching RSS Fetches
Fetch RSS feeds ONCE per cron run, cache in memory, reuse across subscribers.
Don't fetch the same feed 500 times for 500 subscribers.

```javascript
const feedCache = {}

async function getCachedFeed(url) {
  if (feedCache[url]) return feedCache[url]
  const stories = await fetchRSSFeed(url)
  feedCache[url] = stories
  return stories
}
```

## Quote Style Handling
If quoteStyle is 'Surprise Me', pick a random style each day:
```javascript
const QUOTE_STYLES = ['Inspirational','Philosophical','Stoic','Humor & Wit','Historical','Literary','Science & Discovery']

function resolveQuoteStyle(style) {
  if (style === 'Surprise Me') return QUOTE_STYLES[Math.floor(Math.random() * QUOTE_STYLES.length)]
  return style
}
```

## Don'ts
- NEVER use Sonnet or Opus — Haiku only
- NEVER fetch the same RSS feed twice in one cron run
- NEVER send a newsletter if story fetch failed — log and skip
- NEVER include raw RSS HTML in the prompt — strip tags first
